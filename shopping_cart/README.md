# Shopping List CLI App

## Description
A simple command-line shopping list program that allows users to:
- Add items with prices
- View all items
- Calculate total cost

## Features
- Continuous input using while loop
- Stores items and prices using lists
- Displays formatted output
- Calculates total price

## Example Usage
Enter the item you want to buy (press q to quit): Apple  
Enter the price of Apple: 10  
Enter the item you want to buy (press q to quit): Milk  
Enter the price of Milk: 20  
Enter the item you want to buy (press q to quit): q  

----- Shopping List -----  
Apple - $10  
Milk - $20  

The total price is: $30

## Concepts Used
- Lists
- Loops (while, for)
- Conditional statements
